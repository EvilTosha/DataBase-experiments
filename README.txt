Simple distributed database developed during CS-center database course (http://compscicenter.ru/program/course/database2012)

Инструкции по запуску:
1) устанавливаем SBCL, screen:
	 sudo apt-get install sbcl screen libssl1.0.0

4) Для завершения:
   sudo screen -S database-web-console -X quit

Тестировалось на Ubuntu 12.04 x64
